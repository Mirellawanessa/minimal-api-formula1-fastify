# minimal-api-formula1-fastify
Projeto desenvolvido para o desafio da DIO utilizando Node.js e Fastify. A aplicação consiste em uma Minimal API da Fórmula 1 capaz de disponibilizar informações sobre pilotos por meio de endpoints REST. O objetivo é praticar conceitos fundamentais de desenvolvimento backend, criação de rotas, manipulação de dados e estruturação de APIs modernas.
