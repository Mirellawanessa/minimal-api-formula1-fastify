# Minimal API da Fórmula 1 com Node.js e Fastify

Projeto desenvolvido para o desafio da DIO com o objetivo de criar uma API REST simples utilizando Node.js e Fastify.

## Objetivo

Desenvolver uma API capaz de fornecer informações sobre pilotos da Fórmula 1 utilizando uma estrutura minimalista e de fácil manutenção.

## Tecnologias Utilizadas

* Node.js
* Fastify
* JavaScript
* JSON

## Estrutura do Projeto

```text
├── data
│   └── drivers.json
├── routes
│   └── drivers.js
├── server.js
└── README.md
```

## Endpoints

### Listar todos os pilotos

GET /drivers

### Buscar piloto por ID

GET /drivers/
